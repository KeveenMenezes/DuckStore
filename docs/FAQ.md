# FAQ - Perguntas Frequentes

## Sobre GitHub Copilot

### Quando o meu Copilot Pro vence?

Esta pergunta se refere à sua assinatura pessoal do GitHub Copilot Pro. Para verificar quando sua assinatura vence, siga os seguintes passos:

1. Acesse [GitHub](https://github.com)
2. Clique no seu avatar no canto superior direito
3. Selecione **Settings** (Configurações)
4. No menu lateral esquerdo, clique em **Billing and plans** (Faturamento e planos)
5. Procure pela seção **Copilot** onde você verá:
   - O status da sua assinatura (ativa, cancelada, etc.)
   - A data de renovação ou vencimento
   - O método de pagamento

Alternativamente, você pode acessar diretamente: https://github.com/settings/billing

### Recursos do GitHub Copilot Pro

O GitHub Copilot Pro oferece:
- Sugestões de código em tempo real
- Completions de código inteligentes
- Suporte para múltiplas linguagens de programação
- Integração com VS Code, Visual Studio, JetBrains IDEs e outros
- Copilot Chat para assistência interativa

### Como cancelar ou modificar minha assinatura?

Para gerenciar sua assinatura do Copilot Pro:

1. Acesse https://github.com/settings/billing
2. Na seção Copilot, clique em **Edit** (Editar)
3. Você pode:
   - Cancelar a assinatura
   - Atualizar o método de pagamento
   - Alterar o plano
